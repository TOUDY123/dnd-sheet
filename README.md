# D&D 2024 Character Sheet

A fully interactive D&D 2024 Player's Handbook character sheet with:
- Complete character creation (species, background, class, feats)
- Auto-calculated HP, AC, skills, saving throws
- Dice roller with roll history
- Save/load system (5 slots + JSON export/import)
- **Real-time DM Party View** — DM sees all players' stats live using PeerJS P2P

## How to use

1. Open the sheet at your GitHub Pages URL
2. Create your character
3. Enter a room code and click **📡 Share** to broadcast to DM
4. DM enters same room code and clicks **🎲 DM View**

## Hosting

Hosted on GitHub Pages — no server needed for the sheet itself.
PeerJS uses their free cloud for real-time player→DM connections.
